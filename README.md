# PersonApiEmulator
Emulates a person API with realistic Norwegian data

Simulates a population, and exposes the result as two HTTP endpoints

The simulation starts at 1920 and creates 100 initial persons from 1920 and 1940.
Step through one year at a time by pressing enter. It then simulates marriage, divorces, children, death and moving houses, for the next year.

Used for stress testing synchronization
